Spring AI 2.0.0-RC1 has been released as an API stabilization milestone toward the 2.0.0 GA release. The update includes major overhauls to tool calling functionality, improvements to chat memory handling, structured output enhancements, and numerous bug fixes and dependency upgrades across supported AI models.
